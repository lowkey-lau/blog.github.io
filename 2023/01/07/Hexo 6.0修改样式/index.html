Hexo 6.0修改样式

发表于 2023-01-07 更新于 2023-01-09 分类于 博客

2023 年重新抓回写博客的好习惯，发现 hexo 6.0 好像没有暴露出修改的文件

我想使用代码块``来高亮某句文案，但是发现，默认样式实在是太丑了，想着更以前一样修改样式，找了一圈实在没找到样式文件。

修改其实很简单，只需要在根文件夹 > source 创建 _data 文件夹，再创建 styles.styl 文件即可。

例如我当前的代码块样式

code{
    background: #d1939e;
    color #fff;
}
